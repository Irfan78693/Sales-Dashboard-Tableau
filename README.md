# 📊 Sales Dashboard (Tableau)

## 📌 Project Overview

This project presents an interactive Tableau dashboard for analyzing sales performance, revenue trends, and business KPIs.

## 🛠 Tools & Technologies

* Tableau
* Excel / Dataset

## 📊 Key Features

* Sales performance overview
* Region-wise and category-wise analysis
* KPI tracking (Revenue, Profit, Orders)

## 🔍 Key Insights

* Identified top-performing regions contributing major revenue
* Analyzed seasonal sales trends
* Highlighted underperforming categories

* Dataset
